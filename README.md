# react-native-pip-reanimated

PiP view for React Native via react-native-reanimated

## Installation

```sh
npm install react-native-pip-reanimated
```

## Usage

```js
import { multiply } from "react-native-pip-reanimated";

// ...

const result = await multiply(3, 7);
```
